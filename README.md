# MUSTAFA.QA 🛡️
### Integrated Security Management Platform

A comprehensive security operations platform built for real-world deployment.

## Features
- 🚨 **SOS Emergency Button** — instant alert with GPS location
- 📅 **Shift Management** — scheduling, check-in/out, PDF reports
- ⚠️ **Violations Tracking** — record and manage guard violations
- 🧠 **Smart Analytics** — AI-powered pattern detection
- 🎫 **Visitor QR Codes** — auto-generated QR for each visitor
- 🗺️ **Buildings Map** — real-time guard assignment overview
- 📄 **PDF Export** — reports, shifts, and dashboards
- 🔴 **Real-time Alerts** — with emergency siren
- 💬 **Direct Chat** — owner ↔ guard messaging
- ✅ **Full Offline Support** — works without internet

## Tech Stack
- React 19 + TypeScript
- Tailwind CSS v4
- Firebase Firestore (real-time sync)
- jsPDF (PDF export)
- QRCode.js (QR generation)
- Vite (build tool)
- Service Worker (offline support)

## Setup
```bash
npm install
npm run dev
```

## Deploy
```bash
npm run build
# Upload dist/index.html anywhere
```
